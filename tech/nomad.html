ETC Nomad

We use the ETC Nomad attatched to a Touch Screen in the Studio Spaces and for any Fringe shows that are not held in the Theatre

The Nomad uses the same software family as the Ion, only with one output instead of two. It's part of the Eos family (Eos, Gio, Ion, Element), all of these consoles have the same software, so if you've used one you should be able to use them all.

Training

If you've learnt how to use the Ion then the Nomad will not take much to learn. The main difference is the lack of physical buttons. Handily Eos has many Keyboard Shortcuts, which can be found below.

We've also produced a handy quick start guide to using the Nomad, which can be found below.
